üìä Data Binding

Two-Way Binding
<input type="text" data-bind="name">
<div>{name}</div>
Set Data
// JavaScript
setData('name', 'John');
getData('name'); // Returns 'John'
Expressions
<div>{name.toUpperCase()}</div>
<div>{age >= 18 ? 'Adult' : 'Minor'}</div>

üìù Form Elements

Text Input
<input type="text" data-bind="username">
Checkbox
<input type="checkbox" data-bind="agree">
<div>{agree ? 'Yes' : 'No'}</div>
Select
<select data-bind="country">
  <option value="us">USA</option>
</select>
Textarea
<textarea data-bind="message"></textarea>
<div>{message.length} chars</div>
Range
<input type="range" data-bind="volume" 
       min="0" max="100">
Radio
<input type="radio" name="choice" 
       onclick="setData('opt', 'a')">

üß© Components

Define Component
<template component="user-card">
  <div>
    <h3>{name}</h3>
    <p>{email}</p>
  </div>
</template>
Use Component (Static)
<user-card 
  name="John"
  email="john@example.com">
</user-card>
Use Component (Dynamic)
<user-card 
  data-name="userName"
  data-email="userEmail">
</user-card>

üîÑ Iterators

Loop Through Array
<div data-for="users as user">
  <div>{user.name}</div>
  <div>{user.email}</div>
</div>
With Methods
<div data-for="items as item">
  <div>{item.name.toUpperCase()}</div>
  <div>{item.price.toFixed(2)}</div>
</div>
Set Array Data
setData('users', [
  { name: 'Alice', email: 'a@ex.com' },
  { name: 'Bob', email: 'b@ex.com' }
]);
