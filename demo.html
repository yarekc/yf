Features

6C. Component System Features

üí° Key Features:
  • Template-based: Define components using <template component="name">
  • Static Props: Pass values directly like type="success"
  • Dynamic Props: Bind to data using type="{dataKey}" syntax
  • Reactive: Components update automatically when bound data changes
  • Expressions: Use ternary operators and methods inside components
  • Reusable: Define once, use multiple times with different props
‚úÖ Component Syntax:
<!-- Define -->

  <div>{prop1} and {prop2}</div>
</template>

<!-- Use with static values -->

7. Interactive Todo List

A complete application showing multiple features working together

{todos.length} total | {todoCompleted} done | {todoPending} pending
{todo.text}
No tasks yet!

Add your first task above.

todo-app.html 
<!-- Input for new todo -->
<input type="text" data-bind="newTodo">
<button onclick="addTodo()">Add</button>

<!-- Display statistics -->
<p>Total: {todos.length}</p>

<!-- Loop through todos -->
<div data-for="todos as todo">
  <input type="checkbox" onclick="toggleTodo({todo.id}, this.checked)">
  <span class="{todo.done ? 'done' : ''}">{todo.text}</span>
</div>

// JavaScript Functions
<script>
  let nextId = 1;

  function addTodo() {
    const text = getData('newTodo');
    if (!text.trim()) return;

    const todos = getData('todos');
    setData('todos', [
      ...todos,
      { id: nextId++, text, done: false }
    ]);
    setData('newTodo', '');
  }

  function toggleTodo(id, checked) {
    const todos = getData('todos');
    const todo = todos.find(t => t.id === id);
    if (todo) {
      todo.done = checked;
      setData('todos', [...todos]);
    }
  }
</script>
